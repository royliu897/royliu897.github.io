Physics-Informed Deep Learning for Materials

Reconstructing 9D Hamiltonian Parameters at GAMMA Lab

Project Overview

This project focused on developing a surrogate model to predict the scattering intensity of materials based on their Hamiltonian parameters and energy-momentum coordinates. The core challenge was to create a model that was both highly accurate and differentiable, enabling its use in an inverse problem framework to reconstruct 9D Hamiltonian parameters from experimental data.

Key Achievements & Methodology

  • Custom Modulated SIREN Model: Architected a differentiable Modulated SIREN surrogate model with 13 inputs (9 Hamiltonian, 4 energy-momentum). This architecture was chosen for its ability to represent complex, high-frequency signals.
  • 30GB Julia-to-Python Pipeline: Engineered a robust data pipeline to handle the generation of a 30GB dataset (7.5 billion points) using Latin Hypercube Sampling in Julia on TACC supercomputers. I resolved the notoriously difficult Julia-to-Python memory layout interoperability issues to seamlessly transport and transpose data for training in PyTorch.
  • Custom Weighted Loss Function: A critical innovation was the design of a custom loss function. The data is highly sparse, with significant information concentrated in narrow peaks. A standard Mean Squared Error (MSE) loss would cause the model to "collapse" and predict the background, ignoring the peaks. I designed a weighted loss that emphasizes the peaks, preventing this collapse and ensuring high accuracy on the critical features. The loss function was carefully balanced to remain robust against outliers.
  • HPC Training Environment: Configured and managed a high-performance computing environment on the Texas Advanced Computing Center (TACC). I wrote efficient Bash Slurm scripts to handle data loading and distributed training on powerful NVIDIA A100 GPUs.
